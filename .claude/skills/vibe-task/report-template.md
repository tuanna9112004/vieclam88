# Completion Report Template

## Trạng thái

`DONE` | `BLOCKED` | `CHANGES REQUIRED`

## Đã làm

- Kết quả nghiệp vụ đạt được.
- File chính đã sửa.

## Xác minh

| Lệnh | Kết quả |
|---|---|
| `<focused test>` | PASS/FAIL/NOT RUN |
| `<regression/build>` | PASS/FAIL/NOT RUN |

## Rủi ro hoặc blocker

- Chỉ ghi vấn đề còn thật sự ảnh hưởng task.

## Bước tiếp theo

- Tối đa một task nhỏ tiếp theo, không tự triển khai.
